## Requirements

### Build Requirements:

- **Rust** (latest stable version recommended)
  - Install: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
  - Verify: `rustc --version`
- **Cargo** (comes with Rust)

### Runtime Requirements:

- **Wayland compositor** (Hyprland, Sway, etc.)
- **Linux** system :)
